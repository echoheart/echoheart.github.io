# EventHub

- 首先确定eventHub支持的API

	1. 支持on
	2. 支持emit
	3. 支持off
	4. 支持once
	
具体实现请看源码

实现的大体思路

1. on是事件注册, 也就是订阅
2. Emit是事件触发, 也就是发布
3. off解绑事件注册
4. once注册只执行一次的事件
